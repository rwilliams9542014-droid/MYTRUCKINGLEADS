How to Use

1. Search & Enrich: Search for a company name or DOT number to find carriers on otrucking.com and get their email addresses (requires Hunter.io/Apollo API keys configured in .env)

2. Browse by State: See all carriers in a specific state

3. Get Details: Get full information for a specific carrier by DOT number.

4. Batch Search: Search multiple carriers at once

Note: Make sure your backend server is running (npm run dev) and you're logged in for authentication to work.
